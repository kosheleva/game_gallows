# Gallows Game

A funny gallows-game, written in Python. 

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Demo](#demo)

## Installation

The app is using tkinter lib, which is built-in python lib. No other libs installation required.

## Usage

To run a game: 
```sh
python3 main.py
```
